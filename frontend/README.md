# Frontend - CRUD de Professores

Tela simples em HTML, CSS e JavaScript para consumir a API Flask.

## Como executar

Primeiro, execute o backend em:

```text
http://127.0.0.1:5000
```

Depois, entre na pasta do frontend:

```bash
cd frontend
```

Execute um servidor local simples:

```bash
python -m http.server 5500
```

Acesse no navegador:

```text
http://127.0.0.1:5500
```

Também é possível abrir o arquivo `index.html` diretamente no navegador, mas usar o servidor local é mais recomendado.
